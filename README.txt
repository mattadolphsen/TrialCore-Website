TRIALCORE MULTI-PAGE WEBSITE

Open index.html to view the website.

FILES:
- index.html: Home page. This is also the Platform page.
- platform.html: Duplicate platform page for navigation compatibility.
- features.html: Features page.
- safety-compliance.html: Safety & Compliance page.
- about.html: About page.
- contact.html: Contact page.
- styles.css: All colors, spacing, fonts, cards, and layout styles.
- trialcore-dashboard-preview.png: Dashboard image shown on the Home/Platform page.

COMMON EDITS:
- To change text: open the relevant HTML file and edit the words between the tags.
- To change colors: open styles.css and edit the values under :root.
- To replace the dashboard image: replace trialcore-dashboard-preview.png with a new image using the same filename.
